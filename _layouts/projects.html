Quadbot

Quadbot is free software that acts as a chat bot on the popular platform pokemonshowdown.com. It was intially created due to a lack of support for LaTeX on the Pokemon Showdown! servers; yes MathJax also accomplishes the task, but would likely be abused by users to derail conversations, and didn't give power to the users (i.e. me :]). Since then I have added other functionality that I haven't seen on other bots like arbitrary precision arithmetic (that doesn't break), Random Putnam problem generators, Party Parrot generator, a neat periodic parser game (try to construct a word using elements from the periodic table), Markov chain generated text, and a whole lot more. I'm actively making changes to this software, and coordinating with the original container for the bot for bug fixes.

Technologies involved: Python, Asyncio, Image Hostring, Linux CLI, DigitalOcean
