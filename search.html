Use the Search link in the blog toolbar (on the blog index, archive, tags, and each post) to open this page, then use the button below for the search overlay. Search is not in the main site navigation. On this page you can also press Ctrl+K (Windows/Linux) or Cmd+K (Mac) when the search bundle has loaded.

Indexing uses Pagefind on the built site; run ./scripts/docker-index-search.sh locally after a full build, or rely on production deploy.
